# How to Add Images to Your 75 @ 50 Newsletter

## Quick Start

Your newsletter is ready! You just need to add the images from your PDF.

## Step 1: Extract Images from PDF

### Option A: Online Tool (Easiest)
1. Go to https://www.ilovepdf.com/pdf_to_jpg
2. Upload "St Edmunds School and the March of Time.pdf"
3. Download all images

### Option B: Screenshot Method
1. Open the PDF
2. Take a screenshot of each image
3. Save with the names listed below

### Option C: Adobe Reader
1. Open PDF in Adobe Reader
2. Right-click each image â†’ "Save Image As..."

## Step 4: Test Locally

1. Open `75-at-50-newsletter.html` in your web browser
2. All images should now appear!

## Step 5: Upload to GitHub Pages

1. Create a new repository on GitHub
2. Upload all files (the HTML file AND the images folder)
3. Enable GitHub Pages in Settings
4. Your newsletter will be live!

## Tips

- **Image Format**: JPG or PNG both work fine
- **Image Size**: The newsletter will automatically resize images, but try to keep them under 2MB each for faster loading
- **Quality**: Higher quality images will look better, especially for the "Then & Now" comparisons

## Need Help?

If images don't show up:
1. Make sure the image filenames match exactly (case-sensitive)
2. Make sure images are in the `images/` folder
3. Check that the folder structure is correct
4. Try opening the HTML file in a different browser
